# README

* Ruby version: ruby 2.6.3
* Rails version: Rails 6.0.3

* Project set-up:

  1) git clone <this repo>
  2) cd blogsite-api
  3) bundle install
  4) bundle exec guard( to enable guard livereload)

  open another terminal
  
  1) rails db:migrate
  2) rails server
  
 Application will run at http://localhost:3000
